Instalace Mqtt Broker (Mosquitto)

Mosquitto je MQTT broker – komunikační server pro Home Assistant a Zigbee2MQTT. Musí být nainstalován PŘED Zigbee2MQTT! ------------------------------------------------------------------------ 7.1 INSTALACE MOSQUITTO ------------------------------------------------------------------------ KROK 27: Jdi do: Nastavení -> Doplňky -> tlačítko "+ Přidat doplněk" (nebo "ADD-ON STORE") KROK 28: Do vyhledávacího pole napiš: "Mosquitto" -> Klikni na "Mosquitto broker" KROK 29: Klikni na "NAINSTALOVAT" (Install) -> Čekej na dokončení instalace (1–3 minuty) KROK 30: Po instalaci nastavení Mosquitto: -> Klikni na záložku "KONFIGURACE" (Configuration) -> Ponech výchozí nastavení (logins: prázdné = použij HA uživatele) -> ULOŽIT KROK 31: Klikni na záložku "INFORMACE" (Info) -> Přepni "Start on boot" = ON -> Přepni "Watchdog" = ON (automatický restart při pádu) -> Klikni "SPUSTIT" (Start) -> Počkej na zelený stav "Running" ------------------------------------------------------------------------ 7.2 VYTVOŘENÍ MQTT UŽIVATELE V HOME ASSISTANT ------------------------------------------------------------------------ Mosquitto může používat HA uživatele. Doporučuji vytvořit dedikovaného uživatele: KROK 32: Nastavení -> Lidé -> záložka "Uživatelé" -> Klikni "+ Přidat uživatele" -> Jméno: mqtt_user (nebo cokoliv) -> Heslo: [silné heslo – zapiš si ho!] -> Přepni "Správce" = OFF (není správce) -> Přepni "Může se přihlásit" = ON (jen lokální přihlášení) -> ULOŽIT ------------------------------------------------------------------------ 7.3 PŘIDÁNÍ MQTT INTEGRACE DO HOME ASSISTANT ------------------------------------------------------------------------ KROK 33: Nastavení -> Zařízení a služby -> "+ Přidat integraci" -> Vyhledej "MQTT" -> Klikni na "MQTT" KROK 34: HA by mělo automaticky detekovat lokální Mosquitto a nabídnout: "Broker: core-mosquitto" nebo "localhost" -> Broker: localhost (nebo 127.0.0.1 nebo core-mosquitto) -> Port: 1883 -> Uživatel: mqtt_user (nebo jméno které jsi zadal v kroku 32) -> Heslo: [heslo z kroku 32] -> Klikni "ODESLAT" a pak "DOKONČIT" Pokud vše proběhlo správně, MQTT integrace bude zobrazena jako aktivní.
