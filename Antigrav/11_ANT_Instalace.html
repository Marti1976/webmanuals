Instalace — co se děje

Antigravity je aplikace na bázi VS Code, kterou si nainstalujete na lokální počítač. Nic se neposílá do cloudu během instalace.

Co se stane během instalace?

  • Stáhne se balíček (obr. 200-400 MB) z Google serverů
  • Rozbalí se do složky na tvém disku (obvykle Program Files nebo /usr/local)
  • Vytvoří se registrační klíč (Windows) nebo konfigurace (Linux/Mac)
  • Integruje se do systému — můžeš spustit antigravity z terminálu kdekoliv
  • Vytvoří se konfigurační složka (~/.antigravity) v domovské složce

Jak instalace vypadá prakticky

1
Stažení

Stáhneš instalátor z antigravity.google.com/download

2
Spuštění instalace

Dvakrát klikni na .exe (Windows) nebo .dmg/.deb (Linux/Mac)

3
Průvodce instalací

Klikáš "Next" → "Install" → "Finish". Standardní proces.

4
První spuštění

Antigravity se otevře, poprosí tě aby sis přihlásil Google účtem

✓
Instalace je jednoduchá — žádné překvapení

Je to jako instalovat jakýkoliv program. Klikáš, čekáš, hotovo. Žádné skryté složky, žádné chytrosti.

// sekce 02

Kde se co instaluje

Windows

Program Files
├─ Google
│ └─ Antigravity
│ ├─ antigravity.exe
│ ├─ resources
│ └─ ...files...

AppData\Roaming
├─ Google\Antigravity
│ └─ settings.json
│ └─ workspaces
│ └─ extensions

Linux

/usr/local/bin
└─ antigravity

/usr/local/share
└─ antigravity
├─ resources
└─ ...files...

~/.config/antigravity
├─ settings.json
├─ workspaces
└─ extensions

Klíčové složky (důležité pro cleanup)

  • Aplikace — Program Files (Windows) / /usr/local (Linux)
  • Konfigurace & Data — ~/.antigravity (na všech OS)
  • Workspace settings — .antigravity-config v projektu (lokálně v složce kde pracuješ)
  • Extensions — ~/.antigravity/extensions
ℹ️
Skryté složky

Složky začínající tečkou (.) jsou skryté. Aby jsi je viděl v File Exploreru (Windows) nebo Finderu (Mac), musíš zapnout "Show hidden files".

// sekce 03

Ověření instalace

Jak vědět že se Antigravity správně nainstaloval?

Kontrola 1: Antigravity je přístupný z terminálu

$ antigravity --version
Antigravity 1.2.5

✓ Pokud vidíš verzi — OK

Kontrola 2: Aplikace se spustí

$ antigravity
(Měla by se otevřít GUI aplikace)

Kontrola 3: Konfigurace existuje

# Windows:
dir %APPDATA%\Google\Antigravity

# Linux/Mac:
ls ~/.antigravity
✓
Pokud všechny tři kontroly projdou — instalace je OK

Antigravity je připravený na použití.

// sekce 04

Systémové permisí a přístupy

Antigravity si nepoužívá žádné speciální systémové oprávnění, ale měl by mít přístup k:

Co Antigravity potřebuje:

  • Čtení a zápis do souborů v Workspace — tvoje projekty
  • Spouštění příkazů v terminálu — npm, git, build
  • Přístup k Internetu — propojení s Google Gemini API
  • Přístup k Chrome/Chromium — pokud máš extension (screenshot)

Co NEPOTŘEBUJE:

  • Administrátorský přístup (na Windows)
  • root přístup (na Linux)
  • Přístup k systémovým souborům mimo instalační složku
  • Přístup k ostatním uživatelským účtům
⚠️
Nikdy nespouštěj Antigravity "jako správce"

Na Windows to není potřeba a zvyšuje to rizika. Antigravity pracuje bez admin práv.

// sekce 05

Jak odinstalovat Antigravity

Windows — Nastavení Aplikace

1
Otevři Nastavení

Start → Settings → Apps → Installed apps

2
Najdi Antigravity

Hledej "Antigravity" nebo "Google Antigravity"

3
Klikni na tři tečky → Uninstall

Potvrd že chceš odinstalovat. Průvodce udělá zbytek.

4
Restart (pokud se zeptá)

Někdy je potřeba restartovat Windows

Linux — Package Manager

# Debian/Ubuntu:
sudo apt-get remove antigravity
sudo apt-get autoremove

# Fedora:
sudo dnf remove antigravity

Mac — Uninstaller

# Pokud existuje uninstall skript:
/Applications/Google\ Antigravity.app/Contents/MacOS/uninstall

# Nebo tradičně:
rm -rf /Applications/Google\ Antigravity.app
✓
Po odinstalaci:

Hlavní aplikace je pryč. Ale konfigurační soubory a data zůstávají (viz sekce 06).

// sekce 06

Jak očistit všechny zbytky

Po odinstalaci je chytré smazat i konfigurační složky a data.

Windows — Smazání zbytků

# 1. Smaž konfiguraci:
rmdir /S %APPDATA%\Google\Antigravity

# 2. Smaž workspace data:
rmdir /S %APPDATA%\Google\AntgravityData

# 3. Smaž z registry (pokud existuje):
# (To udělá automaticky uninstaller)

Linux/Mac — Smazání zbytků

# Smaž všechny konfigurace:
rm -rf ~/.antigravity
rm -rf ~/.config/antigravity

# Smaž workspace data:
rm -rf ~/.local/share/antigravity

Projekt-specifické soubory (v Workspace)

V každé složce projektu kde jsi s Antigravity pracoval mohou být soubory:

.antigravity-config.json ← smaž pokud chceš
.antigravity-cache/ ← lze smazat, bude regenerována

Ty nejsou kritické — projekty budou fungovat bez nich. Ale čistota je pěkná.

⚠️
NEPŘEKRAČUJ toto!

Nemaž .git, node_modules, src nebo ostatní součásti projektu. Smaž jen Antigravity-specifické věci (`.antigravity-*`).

// sekce 08

Windows vs Linux — důležité rozdíly

Windows

Instalace přes exe + Nastavení

Soubory jsou v AppData

Uninstaluje se přes GUI

Složky jsou skryté — musíš zapnout View Hidden

Linux/Mac

Instalace přes package manager

Soubory jsou v ~/.config

Odinstalace přes apt-get/dnf

Cleanup přes rm -rf (pozor na příkaz!)

Terminál vs GUI

Windows: Uninstalace v GUI je nejjednoduší. Terminálem je zbytečně komplikované.

Linux: Terminálem je nejrychlejší a nejbezpečnější. GUI není vždycky dostupné.

Skryté složky — Linux/Mac je jednodušší

Windows: Musíš zapnout "Show hidden files" v File Exploreru

Linux/Mac: ls -la v terminálu všechno ukazuje

ℹ️
Pro Linux uživatele:

Nedejte si zamést — rm -rf ~/.antigravity je bezpečné. Tečka znamená "home directory", ne root. Smaž jen to.
