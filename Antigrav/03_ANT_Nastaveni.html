Kde jsou settings v Antigravity?

Settings se otevírají na tři způsoby:

  • Klávesová zkratka: Ctrl+, (comma) — stejně jako VS Code
  • Menu: File → Preferences → Settings
  • Икона: V dolním levém rohu (settings gear) → Settings

Otevře se panel se dvěma tabukami:

  • GUI — přátelský interface s checkboxy a slidery (jednodušší)
  • JSON — raw settings.json soubor (pro experty)
💡
Pro začátečníka: Používej GUI tab

Je tam vše jako v normálních aplikacích — checkboxy, dropdown menu, slidery. JSON je pro pokročilé.

Vzhled — Theme (Appearance)

V sekci Appearance najdeš všechno co se týče vizuálu.

Color Theme — barvy editoru

🎨 Dostupné themes

Default Dark — tmavé (doporučeno)

Default Light — světlé

Settings → Appearance → Color Theme → výběr z dropdown

Jak vypadají ikony souboru v levém panelu. Výchozí je "Seti". Můžeš vybrat z několika sad ikon.

// sekce 03

Editor & Kód — jak se chová editor

Tab Size & Indentation

Editor: Tab Size = 2 (nebo 4)
Editor: Insert Spaces = ON
⟹ Když stiskneš Tab, přidá se 2 (nebo 4) mezer
⟹ Tvůj kód bude "zarovnaný" stejně jako ostatní

Proč je to důležité? Tvůj projekt má CSS v Tailwind — pokud budeš mít jiný spacing než ostatní soubory, bude to chaotické.

Word Wrap — zalamování dlouhých řádků

  • Off — řádky se nelamují, musíš scrollovat vpravo
  • On — dlouhý řádek se zalomí do více řádků (čitelnější)

Pro receptář s HTML/JSX components: On je lepší.

Format On Save

⚠️
Editor: Format On Save

Pokud máš nainstalovaný Prettier nebo Eslint: ON
Pokud ne: OFF (aby tě to neštvalo nekončícím formatováním)

Minimap — ten malý náhled vpravo

Minimap ti ukazuje malý náhled kódu vpravo. Pokud je ti to v cestě, vypni: Editor: Minimap Enabled = OFF

Klávesnice — klavesové zkratky

Antigravity používá skoro stejné zkratky jako VS Code. Pokud jsi v tom nový, tady jsou nejdůležitější:

Soubory & Editor

Ctrl+P — Quick Open (najdi soubor)

Ctrl+N — Nový soubor

Ctrl+W — Zavři tab

Ctrl+S — Ulož

Ctrl+Z — Undo

Agent & Chat

Ctrl+K Ctrl+I — Inline prompt (agent v editoru)

Ctrl+L — Clear chat (agent panel)

Ctrl+, — Settings

Ctrl+` — Terminal

Jak si změnit zkratky?

Settings → Keyboard Shortcuts → hledej příkaz → klikni a přeprogramuj.

💡
Tip pro Gemini CLI uživatele

Pokud jsi zvyklý na CLI příkazy a chceš použít agent přes command pallete: Cmd+Shift+P (nebo Ctrl+Shift+P) — otevře command menu kde můžeš psát příkazy Antigravity.

Agentní režimy — nejdůležitější pro tebe

Toto je klíčová volba. Zde nastavíš jak "autonomní" agent bude.

Agent Execution Mode

Najdeš v Settings → Agent → Execution Mode.

🟢 Review-Driven

Nejbezpečnější. Agent plánuje → ty schválíš → agent dělá. Vždy musíš OK dát.

Pro tebe ideální vzhledem k AI Studio traumě

🔵 Agent-Assisted

Středně autonomní. Drobné změny — bez ptaní. Větší věci — čeká. Rozumný kompromis.

🟡 Hands-Off

Více autonomie. Agent pracuje, jen ti hlásí co udělal. Pro zkušené.

🔴 Full Autonomy

Maximálně rizika. Agent si dělá co chce. Nepoužívej pokud máš problém s AI Studiem.

⚠️
Moje doporučení: Začni s Review-Driven

Když se naučíš jak agent funguje a budeš věřit že neudělá něco divného — můžeš přepnout na Agent-Assisted. Ale s Review-Driven nejsi nikdy zaskočen.

Approval Settings

Pokud máš Review-Driven, můžeš ještě nastavit:

  • Auto-approve for small changes — automaticky OK pro změny v jednom souboru (pokud chceš)
  • Show diff before approval — vždy vidíš co se změní (doporučeno ON)
  • Require comments for rejection — když zamítneš plán, musíš říct proč

Chrome Extension — Screenshots & Live Testing

Chrome extension umožňuje agentovi spustit aplikaci v prohlížeči a udělat screenshoty. Je to volitelné, ale velmi užitečné.

Jak nainstalovat?

1. Otevři Chrome Web Store
Hledej "Antigravity Chrome Extension"

2. Klikni "Přidat do Chrome"
Tě poprosí o povolení

3. Ověř v Antigravity Settings
Settings → Extensions → Antigravity Chrome Extension: ON

Co to dělá?

  • Agent může otevřít tvou aplikaci (localhost:3000) v Chrome
  • Udělá automatický screenshot jak to vypadá
  • Vidí co je v HTML — pokud agent chce vědět jak se aplikace chová, ví přesně
  • Spustí formuláře a interakce pokud je to potřeba
✓
Pro receptář je to perfektní

Když agent přidá nový filtr nebo změní komponentu, sám vidí jak to vypadá. Ušetří to spoustu "pokaž mi screenshot" konverzací.

Bezpečnostní poznámka

Extension má přístup k:

  • Obsahu na localhost (tvoje místní aplikace) — v pořádku
  • NEUVIDÍ tvůj obsah na ostatních webech — security je zabudovaná

Pokud nechceš extension — OK. Agent bude pracovat bez něj, jen nebude vidět screenshoty výsledku.

API & Klíče — připojení na Google Gemini API

Antigravity se připojuje k Google Gemini API. Klíč je již nastavený když ses přihlásil svým Google účtem, ale můžeš to změnit.

Kde je to nastavené?

Settings → API & Authentication → Google Gemini API

Možnosti

  • Use Default (Recommended) — Antigravity token, za tebe spravovaný. Nejjednoduché.
  • Use Custom API Key — tvůj vlastní klíč od Google. Pokud máš svůj projekt v Google Cloud Console.
⚠️
Nikdy nesdílej API klíč

Pokud si vytvoříš svůj klíč — nemaž ho do GitHubu ani ho nesdílej s nikým. Klíč by měl být jen v .env lokálně (a v .gitignore).

Rate Limiting — kolik agentů můžeš spustit?

Google má limity pro free tier:

  • 1000 požadavků za den — pro hobby projekt dost
  • 60 požadavků za minutu — agent pracuje dost rychle

Pokud je pro tebe málo — můžeš upgrade na placený Google Cloud.

Workspace — projekty a složky

Antigravity je "multi-workspace" IDE — můžeš mít otevřeno více projektů najednou.

Add Folder to Workspace

File → Add Folder to Workspace → vyběr složky tvého projektu

Pokud máš receptář a chceš přidat jiný projekt na vývoj: File → Add Folder a obě složky se budou zobrazovat v file exploreru.

Workspace Settings

Na úrovni workspace (ne globální) můžeš nastavit:

  • Editor settings pro tenhle projekt — třeba Font Size jen pro receptář
  • Extensions aktivity — které rozšíření se mají v tomhle workspace aktivovat
  • Launch configuration — když spustíš debug, jaké příkazy se spustí
💡
Tip: Workspace File (.code-workspace)

Když máš složitý setup s více projektů, Antigravity vytvoří soubor *.code-workspace který si pamatuje jaké složky byly otevřené. Kdyś ho otevřeš znova — všechno se vrátí do stejného stavu.
